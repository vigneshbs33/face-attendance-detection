1. **Add or upload your face pictures** to the `uploads/` folder
   - Supported formats: `.jpg`, `.jpeg`, `.png`
   - Make sure each image contains a clear, well-lit face
   - Use descriptive filenames (e.g., `john_doe.jpg`, `jane_smith.png`)
2. **Run the registration script** to automatically register all faces:
   ```bash
   python register_uploaded_faces.py
   ```
3. **Verify registration** by checking the "Registered Faces" section in the web interface
